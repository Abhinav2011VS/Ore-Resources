Minecraft Datapack - Download Page

Do you want to convert your precious resources back to ores, for easier storage and transportation? Do you wish you could use different methods to create the ores, such as crafting, cutting, or smelting? Do you want to learn and use the recipes in a simple and intuitive way, with the help of the recipe book and the advancements system? If you answered yes to any of these questions, then this data pack is for you!

This data pack adds recipes to convert the ore resources back to the ores, allowing you to store and transport them more efficiently. You can use the crafting table, the stonecutter, or the blast furnace to create the ores from the resources. The recipes are balanced and compatible with other data packs that add new ores or resources. The data pack also integrates with the recipe book and the advancements system, so you can easily learn and use the recipes.

The data pack supports the following resources and ores:

  • Coal -> Coal Ore
  • Emerald -> Emerald Ore
  • Netherite Scrap -> Ancient Debris
  • Lapis Lazuli -> Lapis Lazuli Ore
  • Gold Ingot -> Gold Ore
  • Gold Nugget -> Nether Gold Ore
  • Quartz -> Nether Quartz Ore
  • Iron Ingot -> Iron Ore
  • Redstone -> Redstone Ore

To install the data pack, simply download the zip file and place it in the datapacks folder of your world save directory. To enable the data pack, use the /reload command or restart your world. To disable the data pack, use the /datapack disable command.
